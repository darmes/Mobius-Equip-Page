READ ME

This script is largely plug-and-play. Open the text file "Mobius's Equip Page Script" and 
copy/paste it into the scripts section. Put it below the default scripts but above Main.
You'll also need to import all the photos in the "Icons" folder. Don't change any names 
or the script won't work.

USAGE

This script is free to use in all games both commercial and non-commercial just 
credit appropiately.
